# EngineeringAgent V2

An agent that thinks before it acts.

Instead of guessing, it maps dependencies.
Instead of retrying blindly, it recovers locally.
Instead of forgetting, it compresses and remembers.

Built on a simple belief: causality is not optional.
Every object must have an origin.
Every result must have a precondition.
Every task must respect the direction of time.

This is not another LLM wrapper.
This is a rule-first execution engine.

## Features

- DAG-based task scheduling
- Causal verification on every output
- Local failure recovery — no full restart
- Three-tier memory compression
- Tool registry with model fallback

Note: Designed for accuracy over speed.
Complex tasks take longer — because every step is verified.
